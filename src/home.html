Crypto PiP

Crypto PiP is a group of smart contract implementations for the Ethereum network. Each smart contract is based on the different ERC standard to illustrate how each one is used in certain use cases. This user screen is created in order to show how a smart contract based on an ERC framework functions through the use of a user interface. The goal is to help the user understand how a smart contract would work in a very easy and straight forward example. The implementation of these smart contracts can also be easily copied and inherited in your own smart contract project.
